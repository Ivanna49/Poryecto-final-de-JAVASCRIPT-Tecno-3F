# CineManíacos 🎬🍿

**¡Bienvenido a CineManíacos!** Un sitio web interactivo sobre películas que consume la API de TheMovieDB.

## ✨ Características Principales

✅ **Diseño Moderno y Responsive**  
- Modo claro/oscuro configurable  
- Adaptable a móviles, tablets y desktop  
- Efectos de hover y transiciones CSS  

✅ **Integración con TheMovieDB API**  
- Películas populares, mejor valoradas y próximos estrenos  
- Búsqueda dinámica  
- Modal con detalles completos  

✅ **Interacción Mejorada**  
- Formulario con validación
